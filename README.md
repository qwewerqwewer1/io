# io

## 👥 Для пользователей
Голосовой чат для общения в реальном времени. Подключайтесь к комнате по ссылке и разговаривайте прямо в браузере без установки приложений.

**Возможности**
- 🎙️ голосовое общение в реальном времени
- 🔗 подключение по ссылке
- 🌐 работа прямо в браузере

## 🛠️ Для разработчиков
Учебный проект для практики real‑time взаимодействия клиента и сервера и подготовки базы для WebRTC‑звонков.

**Стек**
- Vue 3 + Vite
- Node.js
- WebSocket / WebRTC

**Структура**
- `client` — фронтенд
- `server` — бэкенд

**Быстрый старт**
Клиент:
```bash
cd client
npm install
npm run dev
```

Сервер:
```bash
cd server
npm install
node index.js
```

**План**
- обмен событиями через WebSocket
- подключение WebRTC
- передача аудио между пользователями
