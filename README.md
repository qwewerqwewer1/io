# IO — голосовой веб‑чат

Простой голосовой чат, который работает прямо в браузере. Заходите в комнату по ссылке и общайтесь без установки приложений.

## 👥 Для пользователей
- 🎙️ голосовое общение в реальном времени
- 🔗 подключение по ссылке
- 🌐 ничего не нужно устанавливать

## ▶️ Как пользоваться
1. Откройте ссылку комнаты.
2. Разрешите доступ к микрофону.
3. Говорите — вас услышат другие участники.

## ✅ Требования
- современный браузер (Chrome / Edge / Firefox)

## 🛠️ Для разработчиков
Учебный проект, чтобы на практике разобраться в real‑time и WebRTC.

**Стек**
- Vue 3 + Vite
- Node.js
- WebSocket / WebRTC

**Структура**
- `client` — фронтенд
- `server` — бэкенд

**Быстрый старт**
Клиент:
```bash
cd client
npm install
npm run dev
```

Сервер:
```bash
cd server
npm install
node index.js
```

**Переменные окружения**
- `PORT` — порт сервера (по умолчанию 3000)
- `VITE_API_URL` — адрес сервера для клиента

## 🗺️ План
- обмен событиями через WebSocket
- подключение WebRTC
- передача аудио между пользователями
